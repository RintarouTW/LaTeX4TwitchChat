Support LaTeX, plotting, graph, math calculation, highlight code, click to speech etc.. in Twitch Chat.

Click to Speech Language

Note: The option Auto would send the text to the server to tell the language, it may take a while to respond.


The user list to be allowed to show the image automatically by user posted link, for example: your_username, rintaroutw

If you want to enable it for everyone, just enter "*".
